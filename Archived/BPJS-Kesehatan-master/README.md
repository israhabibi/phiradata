# BPJS-Kesehaan
Scrapping Data Faskes, Rumah Sakit, dan klinik yang terdaftar di BPJS Kesehatan.
Data diambil dari website https://lovia.life/id/health/bpjs per Maret 2019.

Tools yang digunakan
1. Python
2. Beautiful Soup
3. Pandas

Data di simpan dalam file CSV
